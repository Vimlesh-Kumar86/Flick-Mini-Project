# Flick - Video Sharing Platform

## Project Overview

Flick is a responsive, static single-page video sharing community website where users can watch embedded YouTube gaming videos, follow popular streamers, sign up, log in, contact the team, and learn about the platform.

Main features:
- Dark/light mode toggle
- Responsive layout (mobile + desktop)
- Sidebar navigation
- Hero section with featured video
- Grid of 50+ gaming YouTube embeds
- About page with team showcase
- Contact form
- Simple signup & login pages

1. Download or clone the project
2. Open `index.html` in any modern browser

No installation or server required.
